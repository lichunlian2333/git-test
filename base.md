Git 是 分布式 版本控制系统
一切反动派都是纸老虎。

vi xxx.file vi 编辑器打开某个文件
vi xxx.file
i 进入编辑模式
esc 退出编辑模式
:wq 保存并推出 vi 编辑器 :q

配置
git config --global user.name "Nicholas"
git config --global user.email "Nicholas@xx.com"

操作
git init
